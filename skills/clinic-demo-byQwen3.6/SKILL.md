# 临床试验数据搜索与导出 Skill

## 任务目的
在 ClinicalTrials.gov 网站上根据指定的疾病条件、研究状态、性别和年龄组进行筛选，勾选目标临床试验记录，并将结果导出为文件。

## 必需参数
- `condition` (string): 目标疾病或条件名称（必填）
- `status` (string): 研究状态筛选条件（选填，默认 Recruiting and not yet）
- `gender` (string): 性别筛选条件（选填，默认 Male）
- `ageGroup` (string): 年龄组筛选条件（选填，默认 Child (birth - 17)）

## 前置条件
- 浏览器可正常访问 `https://clinicaltrials.gov/`
- 无需登录态（公开数据查询）

## 步骤大纲
1. **导航至临床试验网站**：访问 ClinicalTrials.gov 首页。
2. **输入并选择疾病条件**：在搜索框输入疾病名称，从自动补全列表中选择精确匹配项。
3. **选择研究状态并执行搜索**：勾选指定状态，点击 Search 按钮触发查询。
4. **应用性别与年龄筛选**：在结果页侧边栏勾选性别与年龄条件，点击 Apply Filters 刷新列表。
5. **选择目标临床试验记录**：勾选搜索结果列表中的前 3 个临床试验卡片。
6. **导出选中记录**：点击下载图标，在弹窗中确认下载操作。

## Verifier 策略
- 每个步骤均通过 URL 变化、关键文本出现（如 `Clear Filters`、`X selected`）或下载事件触发进行验证。
- 下载步骤通过 `page.waitForEvent('download')` 确保文件流已生成。

## 人工审查注意事项
- 网站动态计数（如 `Male (2,623)`）会随数据更新变化，脚本已参数化文本匹配逻辑。
- 卡片选择逻辑默认选取前 3 项，若需固定特定 NCT ID，需调整 locator 策略。
- 首次执行前请确认网络环境可访问目标站点，并检查浏览器下载目录权限。
